JavaScript Hoisting

During the creation phase, the JavaScript engine moves the variable and function declarations to the top of your code. This feature is known as hoisting in JavaScript. hr
There is 2 types of Hoisting:

  1. Variable Hoisting
  2. Function Hoisting

Variable Hoisting:

Variable hoisting also known as Global hoisting. In this javaScript Engin moves variables on the top of the script.

Function Hoisting

Like variable, Tje javaScript engine moves the variable on the top of the function. And Function Hoisting also known as a Local variable.

Also In Function, if we donot declare the function and invoke/call the function. then it check Function hoisting first and then check in Global Hoisting as well.
